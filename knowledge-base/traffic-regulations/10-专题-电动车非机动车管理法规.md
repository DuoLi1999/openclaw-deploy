# 电动自行车与非机动车管理法规汇编

> 本文汇编电动自行车（电动车）及非机动车相关法律法规
> 整理自：道路交通安全法、电动自行车安全技术规范（GB 17761-2018）等

---

## 一、电动自行车国家标准（GB 17761-2018）

### 强制性标准要求（2019年4月15日起实施）

| 项目 | 标准 |
|------|------|
| 最高车速 | 不超过25km/h |
| 整车质量 | 不超过55kg |
| 电机功率 | 不超过400W |
| 电池电压 | 不超过48V |
| 必须有脚踏骑行功能 | 是 |
| 须具有CCC认证 | 是 |

> **宣传要点**：不符合上述标准的电动两轮车属于电动摩托车（机动车），需持有摩托车驾驶证（D/E/F）、上机动车号牌、购买交强险。无证驾驶电动摩托车按无证驾驶机动车处罚。

## 二、通行规定

### 《道路交通安全法》相关条款

**第五十七条**：驾驶非机动车在道路上行驶应当遵守有关交通安全的规定。非机动车应当在非机动车道内行驶；在没有非机动车道的道路上，应当靠车行道的右侧行驶。

**第五十八条**：残疾人机动轮椅车、电动自行车在非机动车道内行驶时，最高时速不得超过十五公里。

**第五十九条**：非机动车应当在规定地点停放。未设停放地点的，非机动车停放不得妨碍其他车辆和行人通行。

### 禁止行为

根据各地实施办法，电动自行车常见禁止行为包括：
1. 闯红灯
2. 逆向行驶
3. 在机动车道内行驶
4. 醉酒驾驶
5. 搭载超过规定人数（一般限载一名12周岁以下儿童）
6. 驾驶人或乘坐人未佩戴安全头盔
7. 加装遮阳棚、挡风被等影响安全的装置
8. 非法改装（提速、加大电池等）

## 三、处罚标准

### 《道路交通安全法》第八十九条

行人、乘车人、非机动车驾驶人违反道路交通安全法律、法规关于道路通行规定的，处警告或者五元以上五十元以下罚款；非机动车驾驶人拒绝接受罚款处罚的，可以扣留其非机动车。

### 超标电动车按机动车处罚

驾驶不符合国标的电动两轮车（视为机动车）：
- 无证驾驶：《道路交通安全法》第九十九条，罚款200-2000元，可并处15日以下拘留
- 无牌上路：扣留车辆
- 未购买交强险：罚款，扣留车辆

## 四、安全头盔相关

### "一盔一带"安全守护行动（2020年起）

公安部要求：
- 摩托车骑乘人员：必须佩戴安全头盔（法律强制）
- 电动自行车骑乘人员：各地逐步立法强制佩戴

### 记分规定
- 驾驶摩托车不戴安全头盔：记1分（公安部令第163号第十二条第九项）

### 头盔选择标准
合格安全头盔应具有CCC认证标志，符合GB 811-2022《摩托车、电动自行车乘员头盔》标准。

> **宣传数据**：
> - 正确佩戴安全头盔可将交通事故死亡风险降低60%-70%
> - 头部是人体最脆弱且最重要的部位，颅脑损伤是交通事故致死的主要原因
> - 不戴头盔的电动车驾驶人在事故中头部受伤概率是戴头盔者的2.5倍

## 五、电动车消防安全

### 常见规定（各地消防条例）

1. 禁止在建筑物公共门厅、疏散通道、安全出口、楼梯间停放电动自行车或充电
2. 禁止私拉电线从室内向室外为电动自行车充电
3. 禁止在住宅内为电动自行车或其电池充电
4. 应在指定的集中充电设施充电

### 《高层民用建筑消防安全管理规定》（2021年8月1日施行）
第四十七条：在高层民用建筑的公共门厅、疏散走道、楼梯间、安全出口停放电动自行车或者为电动自行车充电，拒不改正的，对经营性单位和个人处2000元以上10000元以下罚款，对非经营性单位和个人处500元以上1000元以下罚款。

> **宣传要点**：电动车起火30秒即可达到不可控状态，100秒内温度可达200°C以上，产生大量有毒烟气。"楼道充电 = 定时炸弹"。

## 六、外卖骑手专项

### 常见违法行为
1. 逆行（赶时间抄近路）
2. 闯红灯
3. 在机动车道穿行
4. 边骑行边看手机接单
5. 超速行驶

### 平台责任
各地交管部门要求外卖平台：
- 建立骑手交通安全培训制度
- 合理规划配送时间，不得以罚款等方式变相鼓励违规
- 配合交管部门通报违法骑手信息
- 为骑手购买意外保险

> **宣传话术**："安全比速度重要""没有一份外卖值得用生命去送""慢一点，安全到"
